# e-commerce-academia-danza
![Estado del Proyecto](https://img.shields.io/badge/Estado-An%C3%A1lisis%20y%20Dise%C3%B1o-blue)
Mejora de un sistema Web para una academia de danza el cual permite conocer los servicios de la academia , asi como un sistema e-commerce de sus productos 

## Objetivos del Proyecto
* **Automatización:** Eliminar procesos manuales de venta y gestión de pedidos.
* **Escalabilidad:** Crear una arquitectura robusta que soporte el crecimiento del catálogo y usuarios.
* **Experiencia de Usuario:** Ofrecer una interfaz moderna, rápida (SSR) y adaptada a dispositivos móviles.

Siguiendo el ciclo de vida de desarrollo de software (SDLC) acualmente se encuentra en fase de analisis y toma de requerimientos:

1.  **[Análisis de Requisitos (en proceso) ](./doc/ERS.md):** Definición detallada de necesidades, requerimientos funcionales y reglas de negocio.


##  Características Principales
- [ ] Landing page para visualizar a la academia.
- [ ] Sección de fotos e interacciòn social.
- [ ] Catálogo dinámico con variantes de producto (talla/color).
- [ ] Carrito de compras persistente.
- [ ] Pasarela de pagos segura con generación de facturas.
- [ ] Panel de administración para control de inventarios.
- [ ] Notificaciones automáticas por correo electrónico.
